## Original brief (Mutual Fund FAQs — Facts-Only Q&A)

Pick ONE product from the following list:

INDMoney
Groww
PowerUp Money
Wealth Monitor
Kuvera.
All Milestones will use the same product you choose here.

### Milestone brief

Build a small FAQ assistant that answers facts about mutual fund schemes—e.g.,
expense ratio, exit load, minimum SIP, lock-in (ELSS), riskometer, benchmark,
and how to download statements—using only official public pages. Every answer
must include one source link. No advice.

### Who this helps

Retail users comparing schemes; support/content teams answering repetitive MF
questions.

### What you must build

- Scope your corpus: Pick one AMC and 3–5 schemes under it.
- Collect 15–25 public pages from AMC/SEBI/AMFI.
- FAQ assistant (working prototype) that answers factual queries only, shows
  one clear citation link in every answer, refuses opinionated/portfolio
  questions with a polite facts-only message and a relevant educational link.
- Tiny UI: welcome line + 3 example questions and a note:
  "Facts-only. No investment advice."

### Key constraints

- Public sources only.
- No PII.
- No performance claims.
- Clarity & transparency.

## Project resolution (binding scope)

The product is **Groww** (per ADR-018, [docs/decisions.md](../../docs/decisions.md)).

| Brief asks | Project decision | Reason |
|---|---|---|
| 3–5 schemes under one AMC | **14 HDFC schemes** in [config/source_urls.json](../../config/source_urls.json) | Stretches the brief to give downstream modules (Review Pulse, Scheduler topics) realistic fund coverage. Still single-AMC. |
| 15–25 public pages | **19 entries** in the manifest plus the static fee explainer (`fee_static_001`): 14 Groww scheme pages + 1 SEBI riskometer FAQ + 2 AMFI knowledge-center pages + 2 Groww help pages | Hits the lower bound of the brief and populates `regulatory_education` and `help_page` content types. |
| One source link per answer | Implemented via `hasRequiredCitations` + the answer prompt rule in [src/rag/answer.ts](../../src/rag/answer.ts). | |
| Refuse advice + educational link | `SAFETY_REFUSAL` in [src/rag/safety.ts](../../src/rag/safety.ts) ends with `https://investor.sebi.gov.in/`. | |
| Tiny UI: welcome + 3 examples + facts-only note | [src/ui/SmartSyncFaqClient.tsx](../../src/ui/SmartSyncFaqClient.tsx) and the unified surface render a welcome, three click-to-fill example chips, and a "Facts-only. No investment advice." badge. | |
| ≤ 3 sentences per answer | **Project upgrades to ≤ 6 concise bullets** to support multi-fund comparisons (matches `docs/rules.md`). The trailer line `"Last updated from sources: <max(last_checked)>"` is appended deterministically by the answer module. | |
| "Last updated from sources: <date>" | Implemented as a deterministic trailer (`ensureTrailer` in `src/rag/answer.ts`). | |
