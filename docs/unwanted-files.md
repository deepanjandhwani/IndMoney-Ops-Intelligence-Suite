# Unwanted / Not Required Files

This list covers files that are useful locally at runtime but are not required in source control, code review, or capstone grading. Do not delete credentials unless you are ready to rerun OAuth.

## Do Not Commit

| Path | Why it is not required |
|---|---|
| `.env`, `.env.local`, `.env.*.local` | Local secrets and machine-specific config. Keep only `.env.example` in git. |
| `credentials/credentials.json` | Google OAuth client secret. Recreate/download from Google Cloud when needed. |
| `credentials/token.json` | User OAuth refresh/access token. Generated by `npm run phase4:oauth-login`. |
| `.chroma/` | Local ChromaDB vector-store persistence generated by `npm run chroma:local` / ingestion. |
| `chroma.log` | Local ChromaDB runtime log. |
| `.data/` | Local SQLite chat-history fallback database. |
| `.vercel/` | Local Vercel project metadata. |
| `artifacts/` | Generated review-pulse/demo exports. |
| `node_modules/`, `.next/`, `out/`, `tsconfig.tsbuildinfo` | Install/build outputs. |
| `__pycache__/`, `*.pyc`, `mcp/__pycache__/` | Python bytecode cache. |
| `.DS_Store` | macOS metadata. |

## Eval Artifacts

| Path | Recommendation |
|---|---|
| `evals/phase3-report.json` | Generated by `npm run evals:phase3` and read by the Admin Evaluation Suite. Keep if you want the dashboard to show the latest run; otherwise regenerate. |
| `evals/REPORT.md` | Canonical generated Markdown report path from `scripts/run-phase3-evals.ts`. |
| `evals/Eval_Report.md` | Non-canonical duplicate name. Rename to `evals/REPORT.md` or remove after confirming the content is already captured. |

## Files To Keep

| Path | Why it is required |
|---|---|
| `.env.example` | Documents all required environment variables without secrets. |
| `config/source_urls.json` | Canonical machine-readable approved source manifest. |
| `config/static_fee_explainer.md` | Approved static fee explainer source for RAG. |
| `docs/source-manifest.md` | Human-readable approved source list. |
| `mcp/groww_ops_mcp_server.py` | Real Google Calendar, Google Sheets, and Gmail draft FastMCP server. |
| `railway.json`, `mcp/Dockerfile` | Deployment config for the MCP sidecar. |
